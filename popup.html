Использовать свои OAuth credentials
Как создать собственные credentials?
  1. Откройте Google Cloud Console
  2. Создайте проект (или выберите существующий)
  3. Включите Gmail API: меню → APIs & Services → Library → найдите Gmail API → Enable
  4. Перейдите в APIs & Services → Credentials
  5. Нажмите + Create Credentials → OAuth client ID
  6. Тип приложения: Desktop app (Настольное приложение)
    Именно Desktop app — он автоматически разрешает http://127.0.0.1. Тип «Web application» не подходит.
  7. Нажмите Create — скопируйте Client ID и Client Secret

⚠️ Если ранее был создан клиент типа Web application — создайте новый типа Desktop app.
