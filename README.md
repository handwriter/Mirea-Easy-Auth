# MireaEasyAuth

Браузерное расширение, которое автоматически извлекает код подтверждения из Gmail и вставляет его на странице входа РТУ МИРЭА — вам больше не нужно переключаться между вкладками.

## Как это работает

1. Вы вводите логин и пароль на странице [sso.mirea.ru](https://sso.mirea.ru).
2. Расширение обнаруживает поле для ввода кода подтверждения.
3. Через Gmail API находит письмо от `sso@mirea.ru`, пришедшее за последние 2 часа.
4. Автоматически вставляет 6-значный код и подтверждает форму.

Весь процесс занимает несколько секунд. Если код не найден в течение 90 секунд — расширение уступает управление, и вы можете ввести код вручную.

## Установка

### Шаг 0 — Скачать файл установки

Перейдите на вкладку **[Releases](../../releases/latest)** репозитория и скачайте нужный файл:

- **`Source code (zip)`** — для Chrome, Edge, Brave
- **`MireaEasyAuth.xpi`** — для Firefox
- **`MireaEasyAuth.app.zip`** — для Safari

---

### Chrome / Edge / Brave


1. Распакуйте скачанный архив `Source code (zip)` в удобное место, например `C:\Extensions\MireaEasyAuth`. **Папку не удаляйте** — браузер будет обращаться к ней при каждом запуске.
2. Откройте страницу расширений в браузере:
   - **Chrome:** `chrome://extensions`
   - **Edge:** `edge://extensions`
   - **Brave:** `brave://extensions`
3. Включите **Режим разработчика** (переключатель в правом верхнем углу).
4. Нажмите **Загрузить распакованное расширение** и выберите распакованную папку (ту, где лежит `manifest.json`).
5. Расширение появится в списке и будет активно.

---

### Firefox

1. Скачайте `MireaEasyAuth.xpi` со вкладки **[Releases](../../releases/latest)**.
2. Откройте **Меню (☰) → Дополнения и темы** или перейдите по адресу `about:addons`.
3. Нажмите на шестерёнку ⚙ → **Установить дополнение из файла…**
4. Выберите скачанный файл `MireaEasyAuth.xpi`.
5. В диалоге подтверждения нажмите **Добавить**.
6. Расширение установлено и активно.

---

### Safari

Расширение распространяется в виде неподписанного `.app` файла. Поскольку приложение не подписано сертификатом Apple, macOS заблокирует его запуск по умолчанию — потребуется снять это ограничение вручную.

1. Скачайте `MireaEasyAuth.app.zip` со вкладки **[Releases](../../releases/latest)** и распакуйте архив.
2. Попробуйте запустить `MireaEasyAuth.app` — macOS покажет предупреждение и откажет в запуске.
3. Откройте **Системные настройки → Конфиденциальность и безопасность**. Внизу появится сообщение о заблокированном приложении — нажмите **Всё равно открыть**.
4. Подтвердите запуск в появившемся диалоге.
5. В Safari откройте **Настройки → Расширения**, найдите MireaEasyAuth и включите его.
6. Разрешите доступ к `sso.mirea.ru` и `mail.google.com`.

> Приложение нужно держать в любом постоянном месте (например, в папке «Программы») — удаление `.app` отключит расширение в Safari.

## Настройка Google OAuth

Расширение использует Google OAuth 2.0 для доступа к Gmail. Есть два способа:

### Способ 1 — Встроенные учётные данные (рекомендуется для большинства)

При первом открытии popup-окна расширения нажмите **«Войти через Google»** — расширение использует предустановленные учётные данные. Никаких дополнительных шагов не требуется.

<!-- SCREENSHOT: popup расширения с кнопкой "Войти через Google" -->

---

### Способ 2 — Собственное приложение в Google Cloud Console

Используйте этот способ, если вы хотите полный контроль над OAuth-приложением или встроенные учётные данные перестали работать.

#### 1. Создайте проект

1. Откройте [Google Cloud Console](https://console.cloud.google.com/).
2. В верхней панели нажмите на выпадающий список проектов → **Новый проект**.
3. Введите название (например, `MireaEasyAuth`) и нажмите **Создать**.

<!-- SCREENSHOT: диалог создания нового проекта в Google Cloud Console -->

#### 2. Включите Gmail API

1. В левом меню выберите **API и сервисы → Библиотека**.
2. В строке поиска введите `Gmail API`.
3. Нажмите на результат **Gmail API** и нажмите **Включить**.

#### 3. Настройте экран согласия OAuth

1. Перейдите в **API и сервисы → Экран согласия OAuth**.
2. Выберите тип пользователей — **Внешний** — и нажмите **Создать**.
3. Заполните обязательные поля:
   - **Название приложения** — любое, например `MireaEasyAuth`
   - **Email службы поддержки** — ваш email
4. Нажмите **Сохранить и продолжить** на всех шагах.
5. На шаге **Тестовые пользователи** добавьте Gmail-адрес, который вы будете использовать для входа.

<!-- SCREENSHOT: шаг "Тестовые пользователи" на экране согласия -->

> **Почему «Внешний» и «Тестовые пользователи»?** Пока приложение не опубликовано, оно работает в тестовом режиме — это нормально для личного использования. Только добавленные тестовые пользователи смогут авторизоваться.

#### 4. Создайте OAuth-учётные данные

1. Перейдите в **API и сервисы → Учётные данные**.
2. Нажмите **+ Создать учётные данные → Идентификатор клиента OAuth**.
3. В поле **Тип приложения** выберите **Desktop app**.
4. Нажмите **Создать**.
5. В появившемся окне скопируйте **Идентификатор клиента** и **Секрет клиента**.

#### 5. Введите учётные данные в расширение

1. Нажмите на иконку расширения в браузере.
2. Раскройте секцию **«Использовать свои OAuth credentials»**.
3. Вставьте **Client ID** и **Client Secret** в соответствующие поля.
4. Нажмите **Сохранить**, затем **«Войти через Google»**.

## Использование без Gmail

Расширение читает письма через Gmail API, поэтому код подтверждения должен приходить именно в Gmail. Если ваш основной почтовый ящик — не Gmail (например, корпоративный, Яндекс, Mail.ru), настройте автоматическую пересылку писем от `sso@mirea.ru`.

### Яндекс Почта

1. Откройте [Яндекс Почту](https://mail.yandex.ru) → **Настройки → Правила обработки писем**.
2. Нажмите **Создать правило**.
3. В условии укажите: **От кого** → `sso@mirea.ru`.
4. В действии выберите **Переслать на адрес** и введите ваш Gmail.
5. Сохраните правило.

<!-- SCREENSHOT: настройка правила пересылки в Яндекс Почте -->

### Mail.ru

1. Откройте [Mail.ru](https://mail.ru) → **Настройки → Фильтры и пересылка → Создать фильтр**.
2. В поле **От** введите `sso@mirea.ru`.
3. В действии выберите **Переслать письмо на адрес** и введите ваш Gmail.
4. Сохраните фильтр.
